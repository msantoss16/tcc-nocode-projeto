Documentação

está documentação é uma manual de uso para seja introduzido ao usuário como funciona os elementos, estilos e do seu projeto.

Elementos

Os elementos são utilizados para compor o seu programa. Todos são baseados nas tags HTML.

Title

HTML: H1
O elemento Title indíca um título no seu programa.

Subtitle

HTML: H3
O elemento Subtitle indíca um subtítulo no seu programa.

Paragraph

HTML: p
O elemento Paragraph indíca um paragrafo no seu programa.

Link

HTML: a
O elemento Link indíca um link no seu programa.

button

HTML: button
O elemento Button indíca um botão no seu programa.

Image

HTML: img
O elemento image indíca uma imagem no seu programa.

Input

HTML: input
O elemento input indíca uma formulario no seu programa.

label

HTML: label
O elemento label indíca um item em uma interface.

Container

HTML: div
O elemento container indíca um container no seu programa.

Formulary

HTML: form
O elemento Formulary indíca um formualrio no seu programa.

SpanDocument

HTML: span
O elemento span é um elemento HTML inline no seu programa.

Board

HTML: table
O elemento board indíca uma tabela no seu programa.

Line

HTML: li
O elemento line indíca uma liha de uma tabela no seu programa.

SvgContainer

HTML: svg
O elemento svg é um contêiner que define um novo sistema de coordenadas e janela de visualização no seu programa.

Timer

HTML: time
O elemento time é um objeto usado para rodar uma função a cada intervalo X de tempo no seu programa.

ProgressTask

HTML: progress
O elemento progress exibe um indicador que mostra o progresso de conclusão de uma tarefa, normalmente exibido como uma barra de progresso no seu programa.

Centralize

HTML: center
O elemento center é usado para alinhar o texto ao centro no seu programa.

DDList

HTML: select
O elemento ddList é usado para criar uma lista suspensa no seu programa.

OptionsList

HTML: option
O elemento O OptionsList define uma opção em um DDList no seu programa.

InlineFrame

HTML: iframe
O elemento InlineFrame é usado para embutir outro documento dentro do documento atual no seu programa.

NavLink

HTML: nav
O elemento NavLink define um conjunto de links de navegação no seu programa.

DocSection

HTML: section
O elemento DocSection define uma seção em um documento no seu programa.

SectionFooter

HTML: footer
O elemento SectionFooter define um rodapé para um documento ou seção no seu programa.

BoardGroup

HTML: tbody
O elemento BoardGroup é usada para agrupar o conteúdo do corpo em uma tabela no seu programa.

BoardCell

HTML: td
O elemento BoardCell Define uma célula de dados em uma tabela no seu programa.

BoardFooter

HTML: tfoot
O elemento BoardFooter é usado para agrupar o conteúdo do rodapé em uma tabela no seu programa.

Bh

HTML: th
O elemento Bh define uma célula de cabeçalho em uma tabela no seu programa.

Bhead

HTML: thead
O elemento Bhead é usado para agrupar o conteúdo do cabeçalho em uma tabela no seu programa.

ColGrp

HTML: colgroup
O elemento ColGrp especifica um grupo de uma ou mais colunas em uma tabela para formatação no seu programa.

ColPrp

HTML: col
O elemento ColPrp especifica as propriedades da coluna para cada coluna dentro de um elemento ColGrp no seu programa.

Dlt

HTML: dl
O elemento Dlt define uma lista de descrição no seu programa.

Dtl

HTML: dt
O elemento Dtl define um termo / nome em uma lista de descrição no seu programa.

Olist

HTML: ol
O elemento Olist define uma lista ordenada no seu programa.

Bline

HTML: tr
O elemento Bline define uma linha em uma tabela no seu programa.

UnorderedList

HTML: ul
O elemento UnorderedList define uma lista não ordenada (com marcadores) no seu programa.

Estilos

Os Estilos são regras de estilo aplicadas aos elementos, são todos baseados em atributos CSS

alinharConteudo

CSS: align-text
Syntax: stretch|center|flex-start|flex-end|space-between|space-around|initial|inherit
Especifica o alinhamento entre as linhas dentro de um contêiner flexível quando os itens não ocupam todo o espaço disponível

alinharItens

CSS: align-self
auto|stretch|center|flex-start|flex-end|baseline|initial|inherit
Especifica o alinhamento para itens dentro de um contêiner flexível

alinhar-se

CSS: align-self
Syntax: auto|stretch|center|flex-start|flex-end|baseline|initial|inherit
Especifica o alinhamento para itens selecionados dentro de um contêiner flexível

tudo

CSS: all
Syntax: initial|inherit|unset
Redefine todas as propriedades (exceto unicode-bidi e direção)

Animação

CSS: animation
Syntax: name|duration|timing-function|delay|iteration-count|direction|fill-mode|play-state
Uma propriedade abreviada para todas as propriedades de animação

Atraso-Animação

CSS: animation-delay
Syntax:time|initial|inherit
Especifica um atraso para o início de uma animação

Direção-Animação

CSS: animation-direction
Syntax: normal|reverse|alternate|alternate-reverse|initial|inherit
Especifica se a animação deve ou não ser reproduzida em reverso em ciclos alternados

Duração-Animação

CSS: animation-duration
Syntax: time|initial|inherit
Especifica quantos segundos ou milissegundos uma animação leva para ser concluída

Preenchimento-Animação

CSS: animation-fill-mode
Syntax: none|forwards|backwards|both|initial|inherit
Especifica um estilo para o elemento quando a animação não está sendo reproduzida (antes de começar, depois de terminar ou ambos)

Contador-Interações-Animação

CSS: animation-iteration-count
Syntax: number|infinite|initial|inherit
Especifica quantas vezes uma animação deve ser reproduzida

Nome-Animação

CSS: animation-name
Syntax: keyframename|none|initial|inherit
Nome da animação

Estado-Animação

CSS: animation-play-state
Syntax: paused|running|initial|inherit
Especifica se a animação está em execução ou pausada

Curva-Velocidade-Animação

CSS: animation-timing-function
Syntax: linear|ease|ease-in|ease-out|ease-in-out|step-start|step-end|steps(int,start|end)|cubic-bezier(n,n,n,n)|initial|inherit
Especifica a curva de velocidade da animação

Visibilidade-Face-Posterior

CSS: backface-visibility
Syntax: visible|hidden|initial|inherit
Define se a face posterior de um elemento deve ou não ser visível quando de frente para o usuário

Plano-de-Fundo

CSS: background
Syntax: color|image|position|size|repeat|origin|clip|attachment|initial|inherit
Uma propriedade abreviada para todas as propriedades de plano de fundo

LigaçãoFundo

CSS: background-attachment
Syntax: scroll|fixed|local|initial|inherit
Especifica se as imagens de fundo são fixas ou rolam com o resto da página

Mesclagem-Fundo

CSS: background-blend-mode
Syntax: normal|multiply|screen|overlay|darken|lighten|color-dodge|saturation|color|luminosity
Especifica o modo de mesclagem de cada camada de fundo (cor / imagem)

Limite-Fundo

CSS: background-clip
Syntax: border-box|padding-box|content-box|initial|inherit
Define até onde o fundo (cor ou imagem) deve se estender dentro de um elemento

Cor-Fundo

CSS: background-color
Syntax: color|transparent|initial|inherit
Especifica a cor de fundo de um elemento

Imagem-Fundo

CSS: background-image
Syntax: url|none|initial|inherit
Especifica uma ou mais imagens de fundo para um elemento

Origem-Imagem-Fundo

CSS: background-origin
Syntax: padding-box|border-box|content-box|initial|inherit
Especifica a posição de origem de uma imagem de fundo

Posição-Imagem-Fundo

CSS: background-origin
Syntax: value
Especifica a posição de origem de uma imagem de fundo

Repetição-Imagem-Fundo

CSS: background-repeat
Syntax: repeat|repeat-x|repeat-y|no-repeat|initial|inherit
Define a repetição da imagem do plano de fundo

Tamanho-Imagem-Fundo

CSS: background-size
Syntax: lenght|auto|cover|contain|initial|inherit
Especifica o tamanho das imagens de fundo

Borda

CSS: border
Syntax: border-width|border-style|border-color|initial|inherit
Uma propriedade abreviada para estilizar a borda de um elemento

Borda-Inferior

CSS: background-bottom
Syntax: border-width border-style border-color|initial|inheritt
Uma propriedade abreviada para estilizar a borda inferior de um elemento

Cor-Borda-Inferior

CSS: background-bottom-color
Syntax: color|transparent|initial|inherit
Uma propriedade abreviada para estilizar a borda inferior de um elemento

Raio-Borda-Inferior-Esquerda

CSS: border-bottom-left-radius
Syntax: length|%|initial|inherit
Define o raio do canto inferior esquerdo

Raio-Borda-Inferior-Direita

CSS: border-bottom-right-radius
Syntax: length|%|initial|inherit
Define o raio do canto inferior direito

Estilo-Borda-Inferior

CSS: border-bottom-style
Syntax: none|hidden|dotted|dashed|solid|double|groove|ridge|inset|outset|initial|inherit
Define o estilo da borda inferior

Largura-Borda-Inferior

CSS: border-bottom-width
Syntax: lenght|medium|thin|thick|initial|inherit
Define a largura da borda inferior

Separação-Borda

CSS: border-collapse
Syntax: separate|collapse|initial|inherit
Define se as bordas da tabela devem ser recolhidas em uma única borda ou separadas

Cor-Borda

CSS: border-color
Syntax: color|transparent|initial|inherit
Define a cor das quatro bordas

Separação-Borda

CSS: border-image
Syntax: source|slice|width|outset|repeat|initial|inherit
Uma propriedade abreviada para todas propriedades de borda de imagem

Imagem-Borda-Inicio

CSS: border-image-outset
Syntax: length|number|initial|inherit
O largura da borda que se estende além da caixa

Repetição-Imagem-Borda

CSS: border-image-repeat
Syntax: stretch|repeat|round|initial|inherit
Especifica se a imagem da borda deve ser repetida, arredondada ou esticada

Corte-Imagem-Borda

CSS: border-image-slice
Syntax: number|%|fill|initial|inherit
Forma de corte da imagem da borda

Raiz-Imagem-Borda

CSS: border-image-source
Syntax: url|none|initial|inherit
O caminho para a imagem a ser usado como borda

Largura-Imagem-Borda

CSS: border-image-width
Syntax: number|lenght|%|auto|initial|inherit
Especifica a largura da imagem da borda
